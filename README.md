# Glitch React Kingcode Tjores en Niels
In deze applicatie heb je de mogelijkheid om als leerling verschillende domeinen, instancies, cursussen en modules in te zien en activities, points challenges, concept challenges en core assignments in te leveren.
Als leeraar heb je de mogelijkheid om nieuwe activities, points challenges, concept challenges en core assignments toe te voegen en goed of af te keuren die door de leerlingen zijn ingeleverd. Verder kun je ook nieuwe domeinen, instancies, cursussen en modules aanmaken en leerlingen wijzigen of verwijderen.
Dit project bestaat uit een webapplicatie en een Android App.

# Applicatie Starten
<strong>Webapplicatie</strong> <br>
Om de webapplicatie te starten moet je de map wp4-2024-react-1e4-kincode-1 zitten. <br/>
Zorg er voor dat docker geinstalleerd is en open staat <br/>
Vervolgens doe je in de terminal: docker-compose -f docker-compose.web.yml up --build <br/>
Nu staat de webapplicatie open op je localhost:3000 <br/>

<strong>Android App</strong> <br/>
Om de Android App te starten heb je ten eerste Android Studio nodig. <br/>
Open Android Studio, ga naar more actions, virtual device manager en open start daar je emulator op. <br/>
Zorg er voor dat docker geinstalleerd is en open staat <br/>
Vervolgens moet je in de map wp4-2024-react-1e4-kincode-1 zitten en doe je in de terminal: docker-compose -f docker-compose.app.yml up --build. <br/>
Nu staat je app open in Android Studio <br/>
